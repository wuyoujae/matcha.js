# 进度条模块 (ProgressBar Module)

### 1. 模块简介
进度条模块位于幻灯片的顶部或底部，通过一条细长的彩条直观地告诉观众：你的演讲已经进行到了什么位置。
它的功能包括：
- **实时同步**：随着翻页动作，进度条会自动伸长或缩短。
- **位置灵活**：可以放在屏幕的最顶端或最底端。
- **样式定制**：支持自定义颜色、高度和动画速度。

---

### 2. 模块使用 demo
进度条通常在初始化时配置一次即可：

```javascript
const app = new Matcha({
  progressBar: {
    color: "#00e676",  // 经典的薄荷绿
    height: "3px",    // 细长的科技感
    position: "top"    // 放在顶部
  }
});
app.start();
```

---

### 3. 模块的语法

#### 初始化配置
在 `new Matcha` 的配置对象中通过 `progressBar` 属性进行设置。

#### 配置属性介绍

| 属性名 | 说明 | 默认值 | 示例值 |
| :--- | :--- | :--- | :--- |
| `visible` | 是否显示进度条 | `true` | `false` |
| `color` | 进度条的颜色 | `"#4a7c59"` | `"#ff5f56"` |
| `height` | 进度条的高度 | `"4px"` | `"2px"` |
| `position` | 显示位置 | `"bottom"` | `"top"` |
| `zIndex` | 层级（防止被遮挡） | `1000` | `9999` |

---

### 4. 介绍模块的其他配置

- **平滑动画**：进度条的变化带有平滑的过渡动画，翻页时不会感到生硬。
- **动态控制**：如果你需要在演讲过程中临时隐藏进度条，可以通过 JS 代码调用 `app.modules.progressBar.hide()` 或 `show()` 来控制。
