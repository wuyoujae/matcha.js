# 数学公式模块 (Math Module)

### 1. 模块简介
数学公式模块让你的幻灯片具备专业的学术表达能力。它集成了强大的 KaTeX 引擎，能够渲染出教科书级别的数学符号。
该模块的作用：
- **公式渲染**：将 LaTeX 语法转化为精美的数学符号。
- **自动加载**：无需手动引入复杂的库，模块会自动从 CDN 获取所需的资源。
- **解析保护**：防止 Markdown 解析器误将公式中的特殊符号（如 `*` 或 `_`）当作格式化标记，确保公式原样显示。

---

### 2. 模块使用 demo
你可以像在 LaTeX 中一样书写公式：

```markdown
# 物理公式示例

行内公式示例：质能方程是 $E = mc^2$。

块级公式示例：
$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$
```

---

### 3. 模块的语法

#### 基本语法
- **行内公式**：使用单个 `$` 包围，如 `$x + y = z$`。
- **独立块公式**：使用双个 `$$` 包围。

#### 支持的语法
该模块支持大部分标准 LaTeX 数学语法，包括但不限于：
- 算术运算符、希腊字母
- 矩阵、行列式
- 积分、求和、极限
- 分式、开方

---

### 4. 介绍模块的其他配置

- **异步重渲染**：由于数学公式库体积较大，模块会在后台异步加载。如果在幻灯片展示过程中库才加载完成，模块会自动重新渲染页面，无需用户刷新。
- **资源定制**：如果你在离线环境使用，可以在 `Matcha` 初始化时通过 `math` 配置项修改 `katexCss` 和 `katexJs` 的指向，改用你本地的资源路径。
- **容错处理**：如果公式书写有误，模块会以红色文字显示错误信息，而不会导致整个幻灯片崩溃。
