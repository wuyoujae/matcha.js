# 样式与主题模块 (Style Module)

### 1. 模块简介

样式模块是 Matcha.js 的“美妆师”。它决定了幻灯片的整体颜色、字体、间距和氛围。
该模块的作用包括：

- **主题切换**：提供多套预设好的精美主题（如极客绿、特斯拉红、深海蓝等）。
- **全局美化**：统一设置标题、正文、卡片的显示样式。
- **自定义风格**：允许你通过简单的指令，为整份幻灯片或某一个单页定制专属的颜色和边距。

---

### 2. 模块使用 demo

你可以通过 Markdown 注释轻松切换主题，或者微调某一页的样式：

```markdown
<!-- 设置整份幻灯片的全局主题为特斯拉风 -->
<!-- global-style: theme=mono -->

# 默认样式的页面

内容...

---

<!-- 这一页想用深海主题，并且文字改成黄色 -->
<!-- theme: ocean -->
<!-- style: text=#ffcc00 -->

# 特殊样式的页面

这一页的风格与众不同。
```

---

### 3. 模块的语法

#### 基本语法

- **全局配置**：`<!-- global-style: 属性=值 -->`（通常放在 Markdown 最开头）。
- **单页主题**：`<!-- theme: 主题名 -->`（放在 `---` 分隔符下方）。
- **单页微调**：`<!-- style: 属性=值, 属性=值 -->`。

#### 样式属性介绍

| 属性名    | 说明                           | 可选值示例                                          |
| :-------- | :----------------------------- | :-------------------------------------------------- |
| `theme`   | 使用预设的主题名称             | `matcha` (默认), `mono`, `ocean`, `sunset`, `light` |
| `bg`      | 页面背景颜色                   | `#000`, `linear-gradient(...)`, `transparent`       |
| `text`    | 正文文字颜色                   | `#fff`, `rgba(255,255,255,0.8)`                     |
| `h1`      | 一级标题颜色                   | `#00e676`                                           |
| `accent`  | 强调色（影响下划线、列表点等） | `#ff0000`                                           |
| `padding` | 页面四周留白级别 (1-10)        | `5` (标准), `10` (极简大留白)                       |
| `gap`     | 布局间距级别 (1-10)            | `5` (标准)                                          |
| `font`    | 字体家族                       | `serif`, `sans-serif`                               |

---

### 4. 介绍模块的其他配置

- **预设尺寸系统**：Matcha 内置了从 1 到 10 的尺寸级别。例如 `padding=10` 会产生非常宽敞的海报感，而 `padding=2` 则适合内容非常密集的页面。
- **动态变量**：样式模块会自动将这些配置转化为 CSS 变量（如 `--slide-bg`），如果你编写自定义组件，可以直接引用这些变量来保持风格统一。
