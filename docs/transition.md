# 过渡动画模块 (Transition Module)

### 1. 模块简介
过渡模块是幻灯片的“润滑剂”。它决定了当你从一页跳转到另一页时，页面是如何切换的。
该模块的作用包括：
- **无缝切换**：让页面之间的跳转告别生硬，变得丝滑顺畅。
- **风格化动画**：提供 3D 翻转、缩放、魔方等多种视觉特效。
- **独立配置**：你可以为整份幻灯片设置默认动画，也可以为特殊的页面定制独特的转场方式。

---

### 2. 模块使用 demo
在页面顶部通过指令设置转场效果：

```markdown
<!-- 设置这一页切入时使用 3D 立方体效果 -->
<!-- transition: cube, duration=1000 -->

# 酷炫的页面
这一页的切换效果非常震撼。

---

<!-- 这一页改用简单的缩放 -->
<!-- transition: zoom -->
# 简洁的页面
```

---

### 3. 模块的语法

#### 基本语法
使用 `<!-- transition: 动画类型, 参数=值 -->` 指令。

#### 动画类型介绍

| 类型名 | 视觉效果描述 |
| :--- | :--- |
| `fade` | 经典的淡入淡出（默认） |
| `slide` | 水平平移切入 |
| `slideUp` | 垂直向上切入 |
| `zoom` | 从中心放大切入 |
| `flip` | 3D 水平翻转 |
| `cube` | 3D 立方体旋转（最酷炫） |
| `none` | 无动画直接切换 |

#### 配置属性介绍

| 参数名 | 说明 | 默认值 | 示例 |
| :--- | :--- | :--- | :--- |
| `type` | 动画类型名称 | `fade` | `cube` |
| `duration` | 动画持续时间（毫秒） | `700` | `1500` (慢动作) |
| `easing` | 动画缓动函数（CSS 语法） | `cubic-bezier(...)` | `ease-in-out` |

---

### 4. 介绍模块的其他配置

- **3D 视距控制**：过渡模块内置了 1200px 的 3D 视距设置，确保 `cube` 和 `flip` 效果具有真实的立体感。
- **全局配置**：你可以在 `Matcha` 初始化时通过 `transition` 配置项设置全局默认的动画类型和时长，这样就不用在每一页都写指令了。
