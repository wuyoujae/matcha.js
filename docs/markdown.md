# Markdown 解析模块 (Markdown Module)

### 1. 模块简介
Markdown 解析模块是 Matcha.js 的“翻译官”。它负责将你编写的纯文本代码翻译成浏览器能看懂的网页结构（HTML）。
除了支持最基础的文字格式化，它还针对幻灯片场景做了特别优化：
- **源码即所得**：你写的换行和空格会被最大限度地保留。
- **增强语法**：在标准 Markdown 之上，增加了对表格、任务列表、彩色文本的良好支持。
- **安全性**：自动处理特殊字符，防止代码注入风险。

---

### 2. 模块使用 demo
你可以使用几乎所有的常用 Markdown 语法：

```markdown
# 一级标题
## 二级标题

这是一段 **粗体文字**，这是一段 *斜体文字*。

- 列表项 A
- 列表项 B

| 表格 | 示例 |
| :--- | :--- |
| 数据 1 | 数据 2 |

> 这是一段精美的引用。
```

---

### 3. 模块的语法

#### 核心语法支持情况

| 语法类型 | 示例写法 |
| :--- | :--- |
| **标题** | `# H1`, `## H2`, `### H3` (最高支持到 H6) |
| **强调** | `**粗体**`, `*斜体*`, `~~删除线~~` |
| **代码** | `` `行内代码` `` 或 ` ```代码块``` ` |
| **列表** | `- 项目` (无序), `1. 项目` (有序), `a. 项目` (字母) |
| **引用** | `> 引用文字` |
| **链接/图片** | `[文字](链接)`, `![描述](图片地址)` |
| **水平线** | `---` 或 `***` |

---

### 4. 介绍模块的其他配置

- **自动段落处理**：模块会自动识别文本段落并加上合适的间距，你不需要手动写 `<p>` 标签。
- **换行保留**：在幻灯片中，你可能希望“回车一次就换一行”。该模块开启了换行保留功能，普通段落中的单个回车也会被转化为可见的换行。
- **代码高亮适配**：生成的代码块会自动带有 `language-xxx` 类名，完美适配各类代码高亮插件。
