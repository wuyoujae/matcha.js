# 布局模块 (Layout Module)

### 1. 模块简介
布局模块是幻灯片的“建筑师”。它能帮你打破单一的上下结构，将页面划分为多列、多行或者网格。
该模块的功能包括：
- **空间划分**：使用极简的符号（如 `+++` 或 `===`）快速分屏。
- **对齐控制**：轻松实现内容居中、靠左、靠右或者顶部对齐。
- **比例分配**：支持自定义每一列或每一行的宽度/高度比例。

---

### 2. 模块使用 demo
使用 `+++` 进行分列，使用 `===` 进行分行：

```markdown
<!-- layout: cols, ratio=1:2 -->
# 左侧标题
+++
# 右侧内容
这里的内容占了三分之二的宽度。

---

<!-- layout: grid, cols=2, rows=2 -->
单元格 1
+++
单元格 2
===
单元格 3
+++
单元格 4
```

---

### 3. 模块的语法

#### 基本语法
在页面顶部使用 `<!-- layout: 布局类型, 参数=值 -->` 指令。
- **分列**：使用 `+++` 分隔内容。
- **分行**：使用 `===` 分隔内容。

#### 布局类型介绍

| 布局名称 | 指令 | 说明 | 分隔符 |
| :--- | :--- | :--- | :--- |
| **居中布局** | `center` | 内容水平垂直居中（默认） | 无 |
| **横分布局** | `cols` | 将页面左右切分为多列 | `+++` |
| **纵分布局** | `rows` | 将页面上下切分为多行 | `===` |
| **网格布局** | `grid` | 类似 Excel 的行列自由组合 | `+++` 和 `===` |
| **文档布局** | `doc` | 适合长文本，左对齐文档流 | 无 |

#### 布局参数介绍

| 参数名 | 说明 | 示例 |
| :--- | :--- | :--- |
| `ratio` | 设置行列比例 | `1:1`, `3:7`, `1:2:1` |
| `valign` | 全局垂直对齐 | `top`, `center`, `bottom` |
| `halign` | 全局水平对齐 | `left`, `center`, `right` |
| `compact` | 紧凑模式（减小间距） | `true` |

---

### 4. 介绍模块的其他配置

- **局部对齐控制**：除了全局设置，你还可以在具体的某一列或某一行内使用 `<!-- align: left -->` 或 `<!-- valign: top -->` 来单独控制该区域的对齐方式，这不会影响其他区域。
- **自动适配**：如果不指定 `ratio`，布局模块会根据内容的多少自动平分空间。
