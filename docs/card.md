# 卡片模块 (Card Module)

### 1. 模块简介
卡片模块是内容的“包装盒”。它能将一段文字或图片包裹在一个精美的容器里，使其在页面中脱颖而出。
该模块的特点：
- **视觉层级**：通过背景、边框和阴影创造出悬浮感。
- **玻璃拟态**：支持目前流行的 `glass` 效果（毛玻璃背景）。
- **自由组合**：卡片内可以包含任何 Markdown 内容，甚至是图片和列表。

---

### 2. 模块使用 demo
使用 `<!-- card -->` 开始，`<!-- endcard -->` 结束：

```markdown
<!-- card: bg=glass, shadow=lg -->
### 核心特性
- 高性能渲染
- 模块化设计
- 极致轻量
<!-- endcard -->

<!-- 简单的彩色卡片 -->
<!-- card: bg=#f0f7ff, border=1px solid #007bff, radius=20px -->
这是一段放在蓝色边框卡片里的重要提示。
<!-- endcard -->
```

---

### 3. 模块的语法

#### 基本语法
- **开启卡片**：`<!-- card: 参数1=值1, 参数2=值2 -->`
- **内容**：中间写正常的 Markdown。
- **关闭卡片**：`<!-- endcard -->`（如果不写，卡片会一直延续到本页结束）。

#### 属性介绍

| 属性名 | 说明 | 示例值 |
| :--- | :--- | :--- |
| `bg` | 背景色或特殊效果 | `#fff`, `glass`, `linear-gradient(...)` |
| `shadow` | 预设阴影强度 | `sm`, `md`, `lg`, `none` |
| `radius` | 圆角大小 | `8px`, `24px` |
| `border` | 边框样式 | `1px solid rgba(0,0,0,0.1)` |
| `padding` | 内边距 | `10px`, `40px` |
| `width` | 宽度 | `400px`, `80%` |

---

### 4. 介绍模块的其他配置

- **自动适应图片**：放在卡片内部的图片会自动获得圆角适配，并且宽度不会溢出卡片。
- **玻璃拟态细节**：当你设置 `bg=glass` 时，模块会自动为你添加模糊滤镜（backdrop-filter）和微弱的白色半透明边框，使其看起来更有质感。
