# 分步展示与高亮模块 (Fragment & Highlight Module)

### 1. 模块简介
这两个模块共同负责幻灯片的“节奏感”。
- **分步展示 (`Fragment`)**：让你的内容不再一次性弹出，而是随着你的点击一段一段地出现。
- **内容高亮 (`Highlight`)**：当你讲到重点时，背景会变暗，指定的文字或图片会像探照灯一样被聚焦放大。
这两个功能可以嵌套使用，创造出极具表现力的讲解效果。

---

### 2. 模块使用 demo
使用 `<!-- step -->` 拆分步骤，使用 `<...>` 聚焦内容：

```markdown
# 讲解步骤
1. 这是第一步
<!-- step -->
2. 这是第二步，接下来看重点：
<!-- step -->
重点是这个 <核心概念>！
<!-- step -->
还有一个 <次要重点>。
```

---

### 3. 模块的语法

#### 分步展示语法
- **基础用法**：直接在两段内容之间插入 `<!-- step -->`。
- **带动画用法**：`<!-- step: 动画类型, duration=时长 -->`。

#### 内容高亮语法
- **语法**：用 `<` 和 `>` 将需要聚焦的内容包围起来。
- **示例**：`请关注这个 <重要的图表>`。

#### 分步动画类型介绍

| 动画名 | 说明 |
| :--- | :--- |
| `fade` | 渐显（默认） |
| `slide-up` | 向上滑入 |
| `zoom` | 缩放进入 |
| `flip` | 翻转进入 |
| `bounce` | 弹跳进入 |

---

### 4. 介绍模块的其他配置

- **嵌套逻辑**：如果你在同一个 `step` 块中写了多个 `<高亮>`，那么点击翻页时，系统会先依次切换这些高亮聚焦，等高亮全部展示完后，才会进入下一个 `step` 块。
- **自动避让组件**：当进入高亮聚焦模式（背景变暗）时，页面上的固定组件（如页码、Logo）会自动降低透明度，以确保观众的注意力集中在当前高亮内容上。
- **自定义时长**：你可以在指令中指定 `duration=1000`（单位毫秒）来让分步动画变慢。
