# 多媒体模块 (Media Modules)

### 1. 模块简介
多媒体模块让你的幻灯片不再只是枯燥的文字。它由图片、视频、音频和网页嵌入四个子模块构成。
这些模块的作用是：
- **增强展示**：支持自定义尺寸、圆角、阴影等视觉效果。
- **自动适配**：图片和视频会自动保持比例，防止变形。
- **智能播放**：支持视频和音频的自动播放、循环、静音等控制。
- **网页集成**：允许你直接在幻灯片中嵌入外部网页或在线演示。

---

### 2. 模块使用 demo
在 Markdown 中使用简单的注释指令插入媒体：

```markdown
<!-- 插入图片：设置宽度为 50%，圆角，带大阴影 -->
<!-- image: src=hero.jpg, size=5, round=lg, shadow=lg -->

<!-- 插入自动循环播放的背景视频 -->
<!-- video: src=bg.mp4, autoplay=true, loop=true, muted=true -->

<!-- 嵌入一个外部网页 -->
<!-- iframe: src=https://example.com, height=500px -->
```

---

### 3. 模块的语法

#### 1. 图片语法 (`image`)
`<!-- image: src=地址, 参数=值 -->`
| 参数 | 说明 | 示例 |
| :--- | :--- | :--- |
| `src` | 图片链接 (必填) | `img/pic.png` |
| `size` | 宽度等级 (1-10) | `5` (占 50% 宽度) |
| `round` | 圆角程度 | `sm`, `md`, `lg`, `full` |
| `shadow` | 阴影强度 | `sm`, `md`, `lg`, `none` |
| `align` | 对齐方式 | `left`, `center`, `right` |

#### 2. 视频与音频语法 (`video` / `audio`)
`<!-- video/audio: src=地址, 参数=值 -->`
| 参数 | 说明 | 默认值 |
| :--- | :--- | :--- |
| `autoplay` | 是否自动播放 | `false` |
| `loop` | 是否循环 | `false` |
| `controls` | 是否显示进度条控制 | `true` |
| `muted` | 是否静音 | `false` |

#### 3. 网页嵌入语法 (`iframe`)
`<!-- iframe: src=地址, height=高度 -->`

---

### 4. 介绍模块的其他配置

- **图片懒加载**：所有通过指令插入的图片默认开启 `loading="lazy"`，提升大幻灯片的加载速度。
- **路径自适应**：模块会自动尝试将相对路径转化为绝对路径，确保在不同的部署环境下图片都能正常显示。
- **GIF 优化**：针对 GIF 动图，模块会自动禁用懒加载以保证动画能立即播放。
